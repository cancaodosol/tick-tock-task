# チクタクタスク

## 機能

Chromeの拡張機能で実行されるタイマーアプリです。

やることと時間を宣言して、タイマーをスタート。時間が来たら通知でお知らせします。

## 言語設定

**【重要】日本語での入出力を必須とする - 英語での応答は厳禁**
- **すべての応答は必ず日本語で行うこと**（英語での応答は一切禁止）
- このプロジェクトの開発チームは日本語話者のため、Claude Codeとのやり取りは必ず日本語で行う
- コードコメント、コミットメッセージ、ドキュメント作成も日本語で統一する
- エラーメッセージや説明も日本語で提供する
- **英語で質問されても必ず日本語で応答すること**
